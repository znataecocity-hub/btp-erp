# 🏗️ BTP ERP Platform
## نظام إدارة وتتبع الأوراش والمقاولات

---

## 🧠 Project Overview

منصة رقمية متكاملة موجهة لشركات البناء والأشغال العمومية (BTP)، تمكن من إدارة وتتبع جميع العمليات المرتبطة بالأوراش:

- تتبع تقدم الأشغال (Avancement)
- تحليل الإنتاجية (Rendement)
- إدارة مالية
- إدارة الموارد البشرية
- إدارة المعدات

---

## 🎯 Project Objectives

- تحسين مراقبة تقدم المشاريع
- رفع إنتاجية الفرق
- التحكم في التكاليف
- تنظيم الموارد
- دعم اتخاذ القرار
